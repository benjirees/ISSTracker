About This Website

What is this?

ISS Tracker

This webpage utilises the 'Wheres the ISS at' API along with the 'Leaflet' map API to display an accurate, real time representation of the International Space Stations location every one second. The 'Wheres the ISS at' API provides a longitude, latitude, altitude and velocity figure allowing the location to be displayed on a world map.

Who made this?

Ben Rees

I am a computer science student who wanted to utilise APIs, I found the 'Wheres the ISS at' API and thought it would be a great opportunity to combine two seperate APIs to make a nice web application that could be used by anyone. Creating this project taught me about using APIs and using them practically to make a working web application.

Technologies Used

HTML, CSS, JS, Third Party APIs

The main webpage of this project is very simple. It uses basic HTML and CSS along with JavaScript to make the APIs work appropriately. Seeing as the main purpose of this webpage is to utilise APIs the HTML and CSS is very basic. However, I used the 'Wheres The ISS at' API which is a rest API and the Leaflet API for the map.
